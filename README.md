# Project Outline
**Project Name**: _Secure Chat App_

## Pages
1. *Login Page*: Offer a _sign up_ & _sign in_ form.
2. *Forgot Password Page*: Offer a Password Reset form, only after validating user details.
3. *Home Page*:
    * Left Pane (25%): Chats List
    * Right Pane (75%): Selected Chat
4. *Account Page*:
    * Left Pane (25%): Setting Category
    * Right Pane (75%): Category specific settings
